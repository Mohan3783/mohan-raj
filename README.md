#  AI Electricity Theft Detection System

## Project Description

This project detects electricity theft using Artificial Intelligence by analyzing abnormal power consumption patterns.

---

## Objectives

* Detect abnormal electricity usage
* Identify theft in real-time
* Automatically cut off power supply
* Provide monitoring through dashboard

---

## Features

* 🔍 AI-based anomaly detection
* ⚡ Real-time monitoring
* 🔌 Auto power cutoff system
* 📱 Mobile/Web dashboard
* 📊 Data visualization

---

## 🛠️ Technologies Used

* Python
* Flask
* Machine Learning (Random Forest / SVM)
* ESP32 (IoT Module)
* HTML, CSS

---

## 🔌 Hardware Components

* ESP32
* Current Sensor (ACS712)
* Relay Module
* Power Supply

---

## 🧠 Methodology

1. Collect power consumption data
2. Train ML model using dataset
3. Detect abnormal patterns
4. Trigger alert / cutoff system
5. Display results on dashboard
